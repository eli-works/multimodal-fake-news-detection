## 0. 标题与摘要

### 0.1 标题

- 基于共享语义桥与双向跨模态注意力的多模态假新闻检测模型
- A Multi-modal Fake News Detection Model with Shared Semantic Bridge and Bidirectional Cross-modal Attention

### 0.2 摘要结构

1. 问题：社交媒体假新闻常同时依赖文本和图像，单模态方法容易遗漏互补信息
2. 方法：提出双路径文本编码、共享语义桥与双向跨模态注意力融合框架
3. 关键组件：
   - BiLSTM + Transformer 双路径文本表示
   - EfficientNet-B0 图像特征提取
   - Shared Bridge 语义对齐
   - BiCrossAttn 细粒度跨模态交互
4. 实验：在 Weibo、Gossip、CFND 三个数据集上验证
5. 结果：完整模型相较常见基线具有更稳定的跨模态建模能力
6. 结论：双路径文本建模与跨模态融合对假新闻检测有积极作用

### 0.3 关键词

- 多模态假新闻检测
- 跨模态融合
- 注意力机制
- LSTM
- Transformer
- EfficientNet

## 1. Introduction

### 1.1 研究背景

- 互联网和社交媒体加速了虚假信息传播
- 假新闻会造成公共认知偏差、社会恐慌和决策误导
- 典型场景包括 COVID-19 谣言、图文不一致内容和伪装成权威语气的误导信息

### 1.2 本文贡献

- 提出双路径文本编码器，兼顾局部上下文与全局语义
- 设计共享残差桥，缓解文本和图像特征分布差异
- 设计双向跨模态注意力模块，增强图文不一致线索捕捉
- 在 Weibo、Gossip、CFND 三个公开数据集上验证有效性

## 2. Related Technologies

### 2.1 文本表示学习技术

- 词向量提供基础语义信息，但难以完整建模上下文
- BiLSTM 擅长捕捉顺序信息和局部上下文
- Transformer 擅长建模长距离依赖和全局语义
- 本文采用 BiLSTM + Transformer 组合文本编码器

### 2.2 图像特征提取技术

- 卷积神经网络可以从局部纹理逐步提取高层视觉语义
- EfficientNet 在精度与计算效率之间具有较好的平衡
- 本文使用预训练的 EfficientNet-B0 作为图像编码器

### 2.3 跨模态表示对齐技术

- 文本和图像特征通常分布在不同语义空间中
- 直接融合容易受到模态差异影响
- 共享残差映射能够在保留原始特征的基础上引入对齐信息

### 2.4 跨模态注意力融合技术

- 多模态任务的核心是建模不同模态之间的互补和冲突关系
- 双向跨模态注意力支持文本到图像、图像到文本的双向信息流
- 这种设计更适合发现图文不一致场景

## 3. Method

### 3.1 模型总览

- 输入文本 `T` 与图像 `I`
- 文本分支：`BiLSTM + Transformer`
- 图像分支：`EfficientNet-B0`
- 融合模块：`Shared Bridge + BiCrossAttn`
- 分类头：`MLP`
- 输出真假标签

### 3.2 文本特征提取模块

- BiLSTM 建模局部上下文和顺序依赖
- Transformer 建模全局语义关系
- 双路径设计保留了短距离和长距离信息

### 3.3 图像特征提取模块

- EfficientNet-B0 作为轻量视觉编码器
- 冻结若干预训练层以保留通用视觉知识
- 输出映射到与文本一致的特征维度

### 3.4 跨模态特征融合模块

#### 3.4.1 共享残差桥

- 目标是缓解模态分布差异
- 通过共享映射增强图文语义对齐

#### 3.4.2 双向跨模态注意力

- 目标是捕捉图文之间的互补与冲突
- 同时支持文本查询图像和图像查询文本

#### 3.4.3 融合拼接

- 对齐后的文本与图像特征拼接
- 送入分类头得到最终预测

### 3.5 预测与损失

- 分类头采用 MLP
- 损失函数采用交叉熵
- 优化方式采用 AdamW

#### 4.4.3 实验结论

- CFND 和 Weibo 上提升更明显

### 4.5 单模态对比

#### 4.5.1 实验设计

- TextOnly：仅使用文本分支
- ImageOnly：仅使用图像分支

#### 4.5.2 关键结果

- 结果建议保留定性结论
- 文本通常是最主要的信息来源
- 图像在部分数据集上能提供互补线索

#### 4.5.3 实验结论

- 文本和图像存在互补关系
- 双模态整体更稳

### 4.6 消融实验

#### 4.6.1 实验设计

| 变体 | 说明 |
| --- | --- |
| Ours | 完整模型 |
| LSTM | 去掉 Transformer 文本路径 |
| TRM | 去掉 BiLSTM 文本路径 |
| -BCA | 去掉 BiCrossAttn |
| -SB | 去掉 Shared Bridge |

#### 4.6.2 关键发现

- 去掉共享桥后，对齐能力下降
- 去掉双向跨模态注意力后，细粒度交互能力下降
- 去掉任一文本路径后，文本表征能力都会变弱

#### 4.6.3 总体结论

- 双路径文本建模、共享桥和跨模态注意力共同支撑模型表现

### 4.7 参数分析

#### 4.7.1 EfficientNet 冻结层数

- 冻结策略会影响图像分支的任务适配程度
- Gossip 对冻结程度更不敏感，CFND 和 Weibo 更敏感

#### 4.7.2 Gossip 的 `max_len` 实验

- 中等长度通常更平衡
- 过短会损失上下文，过长容易引入噪声

### 4.8 案例分析

#### 4.8.1 典型案例

- 案例分析建议展示模型的判断模式，而不是逐条保留置信度数值
- 可以覆盖真新闻、假新闻、假阳性和假阴性几类典型情形

#### 4.8.2 分析结论

- 图文一致时模型更容易做出稳定判断
- 图文冲突时跨模态交互有助于识别异常

#### 4.8.3 改进方向

- 增强困难样本训练
- 做更严格的阈值校准
- 引入更强的事实核验与知识支持能力

- 关键贡献包括：文本双路径建模、共享桥语义对齐和细粒度跨模态交互
