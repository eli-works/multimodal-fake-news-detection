# 实验记录

本文档用于沉淀主实验、对比实验、消融实验和参数分析的可追溯记录。公开仓库中建议保留指标来源、运行脚本、核心配置和结论摘要；完整权重文件可通过 release 或外部链接单独提供。

## 1. 记录规范

每次实验建议记录：

| 字段 | 说明 |
| --- | --- |
| 实验编号 | 如 `main-cfnd-001` |
| 数据集 | `CFND`、`Gossip`、`Weibo` |
| 脚本 | 训练脚本路径 |
| 变体 | `Ours`、`Concat`、`TextOnly`、`ImageOnly`、`-SB`、`-BCA` 等 |
| 关键配置 | `max_len`、`batch_size`、`lr`、`freeze_efficientnet_stages` 等 |
| 输出目录 | `save_root` 下的 run 目录 |
| 指标文件 | `test_metrics.json` 或手动汇总表 |
| 结论 | 一句话描述该实验支撑的论文观点 |

## 4. 消融实验

| 变体 | 对应脚本关键词 | 主要观察 |
| --- | --- | --- |
| `LSTM` | `lstm_only` | 去掉 Transformer 后，全局语义建模能力下降 |
| `TRM` | `transformer_only` | 去掉 BiLSTM 后，局部上下文和顺序线索下降 |
| `-SB` | `wosb` | 去掉共享语义桥后，图文语义对齐能力下降 |
| `-BCA` | `wobicross` | 去掉双向跨模态注意力后，细粒度交互能力下降 |

关键发现：

| 变体 | 影响最大数据集 | 退化幅度 | 原因分析 |
| --- | --- | --- | --- |
| `-SB` | Weibo | F1 -2.83% | 共享桥对图文语义对齐关键 |
| `-SB` | Gossip | 真新闻召回 -25.57% | 少数类识别更依赖跨模态共享线索 |
| `-BCA` | CFND | Precision -5.23% / Recall +4.35% / F1 -0.76% | 缺少细粒度交互后误报增加 |
| `LSTM` | Weibo | F1 -8.23% | 单路径文本建模不稳定 |
| `TRM` | Weibo | F1 -4.41% | 单路径文本建模不稳定 |

## 5. 参数分析

### 5.1 EfficientNet 冻结层数

| 数据集 | 准确率波动 | 解释 |
| --- | --- | --- |
| Gossip | 0.85% | 文本线索较强，图像样式相对单一 |
| CFND | 1.76% | 图像语义与预训练分布较匹配 |
| Weibo | 4.36% | 图像来源复杂，更依赖任务适配 |

### 5.2 Gossip 文本长度

| `max_len` | Accuracy | Fake F1 |
| ---: | ---: | ---: |
| 128 | 0.8618 | 0.9182 |
| 192 | 0.8696 | 0.9225 |
| 256 | 0.8625 | 0.9179 |
| 384 | 0.8562 | 0.9140 |
| 512 | 0.8519 | 0.9108 |
| 1024 | 0.8548 | 0.9133 |
| 1920 | 0.8565 | 0.9142 |

结论：`max_len = 192` 在 Gossip 上表现最好，过长序列会引入噪声并削弱性能。
